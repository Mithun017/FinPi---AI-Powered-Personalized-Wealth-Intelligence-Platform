## Quick Start
1. Clone repo: git clone <repo-url> && cd finpi
2. Set up PostgreSQL: CREATE DATABASE finpi;
3. Configure: cp backend/.env.example backend/.env
4. Add your Gemini API key to backend/.env
5. Run: double-click run.bat
6. Open: http://localhost:3000
7. Login: demo@finpi.ai / Demo@1234

## Environment Variables (backend/.env)
DATABASE_URL=postgresql+asyncpg://user:password@localhost:5432/finpi
SECRET_KEY=your-secret-key-min-32-chars
GEMINI_API_KEY=your-google-gemini-api-key
FRONTEND_URL=http://localhost:3000

## Getting Your Gemini API Key
1. Visit https://aistudio.google.com/app/apikey
2. Click "Create API Key"
3. Paste into backend/.env as GEMINI_API_KEY
